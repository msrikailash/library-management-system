⚡ Quick Setup

This is a Django application — it needs a Python server to run. Follow the steps below to get it running locally or deploy to a free host like Render.

  • 1 Clone the repo: git clone https://github.com/msrikailash/library-management-system-.git
  • 2 Install dependencies: pip install -r requirements.txt
  • 3 Apply migrations: python manage.py migrate
  • 4 Seed sample data: python seed_data.py
  • 5 Start the server: python manage.py runserver then open http://127.0.0.1:8000
